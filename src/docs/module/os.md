<h1>
	Modulo OS
</h1>
<p>
	El modulo os es un modulo que te permite ver
	cierta informacion del sistema operativo donde
	se este ejecutando el programa.
</p>
<ul>
	<li class="gold">
		osname() -> String
		<div class="green">
			Obtener el nombre del sistema operativo
		</div>
	</li>
	<br>
	<li class="gold">
		username() -> String
		<div class="green">
			Obtener el nombre del usuario actual
		</div>
	</li>
	<br>
	<li class="gold">
		run(cmd:String) -> int
		<div class="green">
			Ejecutar un comando. 
			<p>
				-cmd: El comando
			</p>
		</div>
	</li>
	<br>
	<li class="gold">
		kill(pid:int) -> Boolean
		<div class="green">
			Eliminar un proceso (Cerrar programa). 
			<p>
				-pid: Process ID (Identidad del proceso)
			</p>
		</div>
	</li>
	<br>
	<li class="gold">
		getpid() -> int
		<div class="green">
			Obtener el PID del programa de donde fue invocado
		</div>
	</li>
	<br>
	<li class="gold">
		cpu_count -> int
		<div class="green">
			Obtener la cantidad de CPUs del equipo
		</div>
	</li>
</ul>
